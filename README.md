# 🚀 Ilayda Tech Blog

Bu proje ASP.NET Core MVC kullanılarak geliştirilmiş kişisel blog uygulamasıdır.

## 🔧 Kullanılan Teknolojiler
- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- Bootstrap

## ✨ Özellikler
- Admin giriş sistemi (kullanıcı adı / şifre)
- Blog yazısı ekleme, düzenleme, silme (CRUD)
- Blog listeleme ve detay sayfası
- Session tabanlı yetkilendirme

## 🔐 Admin Giriş
- Kullanıcı adı: admin
- Şifre: 1234

## 📌 Amaç
Bu proje, web geliştirme pratiği yapmak ve kişisel bir blog platformu oluşturmak amacıyla geliştirilmiştir.
